Seattle Music Scene Website

Project was created for my PHP final. It is a tribute website celebrating the seattle music scene and its venues

Features:

  • Responsive PHP website with account creation and login page

Covid Date Calculator

  • Added desktop view after google analytics indicated 49% of traffic came from desktops
